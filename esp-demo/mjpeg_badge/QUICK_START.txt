ESP32-S3-LCD-2 MJPEG Badge - Quick Start Guide
==============================================

STEP 1: PREPARE SD CARD
-----------------------
1. Format microSD card as FAT32
2. Copy "output.mjpeg" to the ROOT of the SD card
3. Insert SD card into ESP32-S3-LCD-2 board

STEP 2: ARDUINO IDE SETTINGS
-----------------------------
Tools > Board: "ESP32S3 Dev Module"
Tools > USB CDC On Boot: "Enabled"
Tools > Flash Size: "16MB (128Mb)"
Tools > Partition Scheme: "16M Flash (3MB APP/9.9MB FATFS)"
Tools > PSRAM: "OPI PSRAM"
Tools > Upload Mode: "USB-OTG CDC (TinyUSB)"

STEP 3: UPLOAD
--------------
1. Open "mjpeg_badge.ino" in Arduino IDE
2. Select correct COM port (Tools > Port)
3. Click Upload button
4. Wait for upload to complete

STEP 4: ENJOY!
--------------
Video will play automatically in an infinite loop!

Open Serial Monitor (115200 baud) to see performance stats.

TROUBLESHOOTING
---------------
- "SD Card Error!" → Check SD card is inserted and formatted FAT32
- "File Not Found!" → Ensure output.mjpeg is in SD card ROOT directory
- "Memory Error!" → Make sure PSRAM is enabled in Tools menu
- Can't upload? → Hold BOOT button while connecting USB cable

For detailed help, see README.md
